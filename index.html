About

Reddconomy is a project that brings cryptocurrency into minecraft.
This project is still under development, we advise not to use it with anything but testnets.
Reddconomy is an independent project, we are in no way affiliated with Mojang, Microsoft, Reddcoin or anyone else.

Multicoin support

Despite being originally developed for reddcoin, Reddconomy is blockchain agnostic, as long as the coin provides a fork of bitcoind or exposes compatible RPC apis, it will simply work without any modification.
If required, a BlockchainConnector can be implemented for coins that don't belong to the preceding category.

Everything happens Off-chain

Reddconomy works by creating offchain wallets, where coins are virtually moved while still sitting on a single shared wallet, this ensures immediate in-game transactions and requires zero configuration on the player side.
The player can easily deposit or withdraw coins from an offchain wallet.

Tips

With reddconomy you can easily send and receive tips.

Contract signs

In Reddconomy, offchain contracts are the basis of every in-game transaction, we make them very easy with the use of Contract Signs.
Contract Signs can activate redstone. Contract Signs can also be used to pay. There is another form of Contract Signs, the Temporized Contract Signs, that activate a redstone signal for a specified amount of time. Or disable themselves for the specified amount of time.

Server monetization with deposit, withdraw and transaction fees

Reddconomy lets you set fixed or proportional fees for every action between offchain wallets.

Legend

    • DEBUG: Debug servers are temporary servers used to run untested and sometimes unreleased code and should be considered very unstable
    • TEST: Test servers are temporary servers used to test reddconomy public releases
    • GAME/ vanilla: Normal minecraft servers that use reddconomy.
    • GAME/ mods: Modded minecraft servers that use reddconomy.
  • COIN
    • */ testnet: The server uses testnet coins. Testnet coins works exactly like real coins but have no value and are usually much easier to obtain.
