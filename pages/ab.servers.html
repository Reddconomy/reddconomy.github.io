Legend

    • DEBUG: Temporary server used to run untested and often unreleased code. Very unstable.
    • TEST: Server used to test reddconomy public releases
    • GAME/ vanilla: Minecraft server which uses reddconomy.
    • GAME/ mods: Modded minecraft server which uses reddconomy.
  • COIN
    • */ testnet: Server which uses testnet coins. Testnet coins work exactly like real coins but have no value and are usually much easier to obtain.
